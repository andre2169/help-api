#  Sistema de Chamados help – API

Este projeto é uma **API REST para gerenciamento de chamados de TI**, desenvolvida com foco em aprendizado prático de:

- Arquitetura de APIs
- Sistemas distribuídos
- Mensageria assíncrona
- Boas práticas de backend

O projeto está sendo construído **incrementalmente**, com versionamento desde o início.

---

## Tecnologias (em uso e planejadas)

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- RabbitMQ
- Docker

---

## Arquitetura (em evolução)

- API REST para abertura e gestão de chamados
- Mensageria para processamento assíncrono de eventos
- Separação clara de camadas (API, serviços, banco e mensageria)
